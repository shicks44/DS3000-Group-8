Why This Is Required
____________________________________________________________________________________________________________________________________________________________________________

GitHub’s notebook renderer cannot display Colab widget metadata (e.g., application/vnd.jupyter.widget-view+json).
This causes GitHub to show an “Invalid Notebook” preview.
This is normal and does not indicate any problem with the notebook itself.

Everything runs correctly once opened in Colab.
